# 📊 BTC & XAUT Daily Report Bot

Телеграм-бот, который каждый день в **10:00 МСК** присылает отчёт по **Bitcoin (BTC)** и **Tether Gold (XAUT)**.

Отчёт содержит:
- 💰 Текущую цену
- 📅 Изменение за день и за неделю
- 🔮 Прогнозы: краткосрочный / среднесрочный / долгосрочный

---

## 🚀 Деплой на Railway (шаг за шагом)

### 1. Подготовь токены

**Telegram-токен:**
1. Открой [@BotFather](https://t.me/BotFather)
2. `/newbot` → придумай имя → скопируй токен

**Твой Chat ID:**
1. Открой [@userinfobot](https://t.me/userinfobot)
2. Нажми `/start` — бот пришлёт твой `id` (число вида `123456789`)

**Anthropic API Key:** уже есть у тебя ✅

---

### 2. Залей код на GitHub

```bash
git init
git add .
git commit -m "init"
git remote add origin https://github.com/ТВО_ИМЯ/btc-xaut-bot.git
git push -u origin main
```

---

### 3. Создай проект на Railway

1. Зайди на [railway.app](https://railway.app) → **New Project**
2. Выбери **Deploy from GitHub repo** → выбери репозиторий
3. Railway автоматически обнаружит `railway.toml` и `Procfile`

---

### 4. Задай переменные окружения

В Railway → твой проект → вкладка **Variables** → добавь три переменные:

| Переменная | Значение |
|---|---|
| `TELEGRAM_TOKEN` | токен от @BotFather |
| `ANTHROPIC_API_KEY` | твой ключ Anthropic |
| `CHAT_ID` | твой chat_id из @userinfobot |

---

### 5. Деплой

Railway задеплоит бота автоматически после добавления переменных.
Проверь вкладку **Logs** — должна появиться строка:
```
🤖 Бот запущен. Отчёт каждый день в 07:00 UTC (10:00 МСК).
```

---

## ⏰ Часовой пояс

Бот работает на UTC. Настройки в `bot.py`:
```python
DAILY_HOUR   = 7   # 07:00 UTC = 10:00 МСК
DAILY_MINUTE = 0
```

Если нужно другое время — измени `DAILY_HOUR`.
